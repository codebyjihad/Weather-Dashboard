# Weather Dashboard

A modern and responsive Weather Dashboard built with **React**, **Redux Toolkit**, and **Tailwind CSS**, fetching live weather data from [OpenWeatherMap API](https://openweathermap.org/).

---

## Features

- Search weather by city name  
- Live weather information: temperature, description, and city name  
- Responsive and beautiful UI using **Tailwind CSS**  
- Weather icons based on current weather conditions using **react-icons**  
- Loading and error handling  
- Clear weather data button  

---

## Technologies Used

- **React** – Frontend framework  
- **Redux Toolkit** – State management  
- **Tailwind CSS** – Styling  
- **React Icons** – Weather icons  
- **OpenWeatherMap API** – Weather data
